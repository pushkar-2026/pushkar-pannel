**Fields**: `key` (String)\n- **Usage**: Sent to `/api/login` endpoint\n\n#### `LoginResponse.java`\n- **Purpose**: Response model from login API\n- **Fields**: `success`, `message`, `error`, `user`\n- **Usage**: Parsed response from `/api/login`\n\n#### `User.java`\n- **Purpose**: User data model\n- **Fields**: `id`, `username`, `role`, `balance`\n- **Usage**: Stored in SharedPreferences after login\n\n#### `ApiResponse.java`\n- **Purpose**: Generic response for other endpoints\n- **Fields**: `success`, `valid`, `message`, `error`, `balance`, `validityType`, `expiresAt`\n- **Usage**: Used by `/check_key`, `/use_key`, `/api/stats`\n\n---\n\n### API Layer (`api/`)\n\n#### `ApiService.java`\n- **Purpose**: Retrofit interface defining all API endpoints\n- **Methods**:\n  - `loginWithKey()` - POST /api/login\n  - `checkServerStatus()` - GET /apk_connect\n  - `checkKey()` - POST /check_key\n  - `useKey()` - POST /use_key\n  - `getStats()` - POST /api/stats\n\n#### `RetrofitClient.java`\n- **Purpose**: Singleton for Retrofit instance\n- **Features**:\n  - OkHttp client configuration\n  - HTTP logging interceptor\n  - Connection timeouts\n- **Usage**: `RetrofitClient.getClient().create(ApiService.class)`\n\n#### `ApiClient.java`\n- **Purpose**: Wrapper around Retrofit with callback handling\n- **Methods**:\n  - `loginWithKey(key, callback)`\n  - `checkServerStatus(callback)`\n  - `checkKey(key, callback)`\n  - `useKey(key, callback)`\n- **Callback Interface**: `ApiCallback<T>` with `onSuccess()` and `onError()`\n\n---\n\n### Activities (`activities/`)\n\n#### `LoginActivity.java`\n- **Purpose**: Login screen\n- **Features**:\n  - License key input\n  - Login button with validation\n  - Progress indicator\n  - Auto-redirect if already logged in\n- **Flow**: Input key → API call → Save user → Navigate to MainActivity\n\n#### `MainActivity.java`\n- **Purpose**: Main app screen\n- **Features**:\n  - Display user info (username, balance, role)\n  - Key activation input\n  - Logout button\n  - Auto-redirect if not logged in\n- **Flow**: Show user data → Input key → Activate → Update balance\n\n---\n\n### Utilities (`utils/`)\n\n#### `Constants.java`\n- **Purpose**: Centralized configuration\n- **Contains**:\n  - `BASE_URL` - Server URL\n  - API endpoints\n  - SharedPreferences keys\n  - Error/success messages\n  - Timeout values\n\n#### `SharedPrefManager.java`\n- **Purpose**: Local data storage management\n- **Methods**:\n  - `saveUser(user)` - Save user after login\n  - `getUser()` - Retrieve saved user\n  - `isLoggedIn()` - Check login status\n  - `updateBalance(balance)` - Update user balance\n  - `clearAll()` - Clear all data (logout)\n\n---\n\n### Layouts (`layouts/`)\n\n#### `activity_login.xml`\n- **Components**:\n  - Header text (\"ASHWIN VIP PANNEL\")\n  - License key EditText\n  - Login button\n  - Progress bar\n  - Footer info text\n\n#### `activity_main.xml`\n- **Components**:\n  - Welcome header\n  - User info card (username, balance, role)\n  - Key activation section\n  - Key input EditText\n  - Activate button\n  - Progress bar\n  - Logout button\n\n---\n\n## 🔄 Data Flow\n\n### Login Flow\n```\nLoginActivity\n    ↓\nUser enters key\n    ↓\nApiClient.loginWithKey()\n    ↓\nRetrofitClient → ApiService.loginWithKey()\n    ↓\nPOST /api/login\n    ↓\nLoginResponse received\n    ↓\nSharedPrefManager.saveUser()\n    ↓\nNavigate to MainActivity\n```\n\n### Key Activation Flow\n```\nMainActivity\n    ↓\nUser enters key\n    ↓\nApiClient.useKey()\n    ↓\nRetrofitClient → ApiService.useKey()\n    ↓\nPOST /use_key\n    ↓\nApiResponse received\n    ↓\nSharedPrefManager.updateBalance()\n    ↓\nUI updates with new balance\n```\n\n---\n\n## 🔐 Security Features\n\n1. **Session Management**: Cookies handled by OkHttp\n2. **Local Storage**: SharedPreferences for user data\n3. **HTTPS**: All API calls use HTTPS\n4. **Error Handling**: Proper error messages and logging\n5. **Input Validation**: Key format validation before API calls\n\n---\n\n## 📦 Dependencies\n\n```gradle\n//
